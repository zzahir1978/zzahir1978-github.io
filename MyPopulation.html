Malaysian Population

Exploratory Data Analysis Based on Malaysian Population Dataset

Dataset: Malaysia Population (000), 1891 2015

This data set shows the number of population in thousand (000).
The url for the data set are as follow: http://data.stats.gov.my/download_bppd.php?cat=3&id=1

Incorporating the dataset into R.

dataset_a <- read.csv("/Users/zzahir1978/Desktop/Sample data/bppd_Population_of_Malaysia_1891-2015.csv")
dataset_a <- rename(dataset_a, Population=Total...000.)
